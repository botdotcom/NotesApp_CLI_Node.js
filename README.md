## (CLI) Notes App - Node.js

Run this application as follows:
1. To add a new note
`node app.js add --title="{note_title}" --body="{body}"`

2. To remove a note
`node app.js remove --title="{note_title}"`

3. To read a note
`node app.js get --title="{note_title}"`

4. To list all notes
`node app.js list`
